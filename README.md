# Social-Media-Impact-Analysis
Analyzing the correlation between social media usage and mental health among students using Python.
Social Media & Mental Health Data Analysis

## Project Overview
This project investigates the relationship between social media usage, addiction levels, sleep patterns, and the mental health of students aged 18-24. Using a data-driven approach, I identified the primary digital drivers of psychological well-being.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
* **Models:** K-Means (Clustering), Random Forest (Feature Importance)

## Key Findings
* **The "Conflict" Factor:** The analysis revealed that "Conflicts Over Social Media" is the strongest predictor of negative mental health outcomes, even more than total usage time.
* **Sleep Impact:** Identified a clear correlation between high addiction scores and disrupted sleep cycles.
